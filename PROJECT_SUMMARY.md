## 关键算法实现

### 1. PowerMem时间衰减算法

```javascript
最终评分 = 相似度 × 0.5 + 重要性 × 0.3 + 新鲜度 × 0.2
新鲜度 = exp(-0.1 × 天数)
```

### 2. 余弦相似度计算

```javascript
similarity = dotProduct(A, B) / (magnitude(A) × magnitude(B))
```

### 3. Token估算

```javascript
tokens = 中文字符数 × 1.5 + 英文单词数 × 1.3 + 标点符号数 × 1
```

### 4. 自动压缩触发

```javascript
if (tokenUsage > maxTokens × 0.8) {
  compressOldConversations();
  rebuildContext();
}
```

## 已知限制

### 1. API相关
- **Embedding支持**：Grok API可能不支持embedding端点，已实现降级方案
- **流式输出**：部分API可能不支持SSE，会自动降级到非流式
- **API Key暴露**：纯前端项目无法完全隐藏API Key

### 2. 功能限制
- **离线模式**：需要网络连接才能生成内容
- **数据同步**：暂不支持跨设备同步
- **协作功能**：暂不支持多人协作

### 3. 性能限制
- **大量记忆**：超过1000条记忆可能影响检索速度
- **长对话**：超过100轮对话可能影响加载速度
- **浏览器限制**：IndexedDB存储受浏览器配额限制
