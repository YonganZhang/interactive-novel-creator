# 🚀 快速开始指南

欢迎使用交互式小说创作系统！这份指南将帮助你在5分钟内开始创作。

## 📋 准备工作

### 检查清单
- [ ] 已安装Python 3.x（检查：在命令行输入 `python --version`）
- [ ] 浏览器版本：Chrome 90+ / Edge 90+ / Firefox 88+
- [ ] 网络连接正常（用于AI生成内容）

## 🎬 第一步：启动服务器

### Windows用户
双击运行 `启动服务器.bat`

### Mac/Linux用户
```bash
cd /path/to/小说
python3 -m http.server 8080
```

看到这样的提示就成功了：
```
Serving HTTP on :: port 8080 (http://[::]:8080/) ...
```

## 🌐 第二步：打开浏览器

在浏览器地址栏输入：
```
http://localhost:8080
```

## 🎮 第四步：开始创作

### 输入你的第一个行动
在输入框中描述主角的行动，例如：
```
我走出村庄，踏上了寻找父母的旅程。
```

### 等待AI生成
AI会根据你的世界观和行动生成后续剧情，通常需要5-10秒。

### 继续对话
根据AI的描述，继续输入主角的行动，推进剧情发展。

## 📚 第五步：管理记忆

### 打开记忆面板
点击左上角的 📚 按钮

### 查看记忆
- **全部**：查看所有记忆
- **角色**：查看角色信息
- **场景**：查看场景描述
- **事件**：查看重要事件

### 添加记忆
1. 点击"+ 添加记忆"按钮
2. 选择类型（角色/场景/事件等）
3. 填写标题和内容
4. 设置重要性（1-10）
5. 点击保存

**示例记忆**：
```
类型：角色
标题：神秘商人
内容：一个穿着黑色斗篷的商人，总是出现在关键时刻，似乎知道很多秘密。
重要性：7
```

## 💡 使用技巧

### 1. 描述要具体
❌ 不好：我去了森林
✅ 好：我小心翼翼地走进幽暗的森林，手握着父亲留下的短剑

### 2. 添加关键记忆
当出现重要角色、地点或事件时，立即添加到记忆库：
- 新角色出现 → 添加角色记忆
- 到达新地点 → 添加场景记忆
- 重要转折 → 添加事件记忆

### 3. 定期查看Token使用
右下角显示Token使用情况：
- 绿色（<70%）：正常
- 橙色（70-85%）：接近上限
- 红色（>85%）：即将自动压缩

### 4. 利用菜单功能
点击右上角 ⋮ 按钮：
- **导出对话**：保存你的故事
- **手动压缩**：清理旧对话
- **切换项目**：管理多个故事
- **创建新项目**：开始新故事

## ⚠️ 常见问题

### Q1：AI生成失败怎么办？
**A**：检查网络连接，刷新页面重试。如果持续失败，查看浏览器控制台（F12）的错误信息。

### Q2：如何保存我的故事？
**A**：数据自动保存在浏览器中。也可以通过"导出对话"功能保存为文本文件。

### Q3：可以创建多个故事吗？
**A**：可以！点击菜单 → 创建新项目，可以管理多个独立的故事。

### Q4：记忆太多会影响性能吗？
**A**：系统会自动检索最相关的5条记忆，不会全部加载。建议保持在100条以内。

### Q5：Token用完了怎么办？
**A**：系统会自动压缩旧对话。你也可以手动压缩，或者创建新会话。

## 📱 移动端使用

### 在手机上访问
1. 确保手机和电脑在同一WiFi
2. 查看电脑IP地址（Windows: `ipconfig`，Mac: `ifconfig`）
3. 在手机浏览器输入：`http://电脑IP:8080`

### 添加到主屏幕
1. 在手机浏览器打开网站
2. 点击浏览器菜单
3. 选择"添加到主屏幕"
4. 像App一样使用

## 🎓 进阶技巧

### 1. 使用标签关联记忆
在记忆内容中提到其他角色或地点，系统会自动建立关联。

### 2. 设置记忆重要性
- **10分**：主角、核心设定
- **8-9分**：重要配角、关键剧情
- **5-7分**：次要角色、普通事件
- **1-4分**：背景信息、琐碎细节

### 3. 定期整理记忆
删除不再需要的记忆，保持记忆库整洁。

### 4. 利用压缩摘要
查看压缩摘要可以快速回顾之前的剧情。
