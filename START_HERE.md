## 📱 移动端使用

### 在手机上访问
1. 确保手机和电脑在同一WiFi
2. 查看电脑IP地址
3. 手机浏览器输入：`http://电脑IP:8080`

### 添加到主屏幕
1. 在手机浏览器打开网站
2. 点击浏览器菜单
3. 选择"添加到主屏幕"
4. 像App一样使用

## ⚠️ 重要提示

### 必须注意
1. **需要网络连接** - AI生成需要调用API
2. **使用HTTP服务器** - 不能直接打开HTML文件
3. **浏览器要求** - Chrome 90+ / Edge 90+ / Firefox 88+
4. **定期备份** - 使用导出功能保存故事

### 建议注意
1. **API Key安全** - 纯前端项目会暴露API Key
2. **浏览器存储** - 不要清除浏览器数据
3. **记忆管理** - 定期清理不需要的记忆
4. **Token监控** - 注意Token使用情况

## 🐛 遇到问题？

### 常见问题
1. **页面空白** - 检查浏览器版本和控制台错误
2. **API调用失败** - 检查网络连接和API Key
3. **数据丢失** - 检查是否清除了浏览器数据
4. **记忆检索不准** - 增加记忆重要性或内容详细度

### 排查步骤
1. 打开浏览器控制台（按F12）
2. 查看Console标签的错误信息
3. 查看Network标签的网络请求
4. 参考 `README.md` 的故障排除章节
