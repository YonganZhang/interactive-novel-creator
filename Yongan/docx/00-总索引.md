## 项目关键信息

**API 配置**：
- 平台：智增增平台（xAI Grok）
- 模型：grok-4-1-fast-reasoning
- API Key：预填默认值 + 用户可自改（localStorage）

**登录信息**：
- 用户名：admin / 密码：admin123
- 验证方式：SHA-256 哈希比对
- 会话：sessionStorage（关闭浏览器自动登出）

**核心算法**：
- PowerMem 时间衰减：`最终评分 = 相似度×0.5 + 重要性×0.3 + 新鲜度×0.2`
- 自动压缩阈值：80% (6400/8000 tokens)

**数据存储**：
- 数据库：IndexedDB (`NovelCreationDB`)
- 5个数据表：projects、memories、conversations、sessions、summaries
